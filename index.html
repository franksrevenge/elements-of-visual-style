Elements of Visual Style

Pure JavaScript visual effects implemented on HTML5 Canvas.

The purpose of this project is to implement various 2D and 3D visual effects using HTML5 Canvas and without the help of any external libraries or WebGL.

See source at GitHub.

Effects

  • 3D projection
  • Bump mapping
  • Bilinear interpolation
  • Fisheye lens
  • Crepuscular rays
  • Plasma effect

Fractals

  • Mandelbrot

    • Primitives

    • Line
    • Solid triangle
    • Interpolated triangle
    • Textured triangle
